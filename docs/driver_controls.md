# 🚗 Driver Controls — LER-2026

**Controller:** Xbox Controller (Port 0)

| Input | Action |
|---|---|
| **Left Stick** | Translate (forward / back / strafe) |
| **Right Stick X** | Rotate |
| **Right Trigger** | Precision drive (speed modifier) |
| **A Button** | Reset gyro (re-zero field orientation) |
| **B Button** *(hold)* | Face hub (auto-rotate toward hub while strafing) |

> **Deadband:** 0.1 on all axes. Drive scaled to 75%, rotation scaled to 3×.
